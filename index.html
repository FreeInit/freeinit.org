FreeInit.org is the community initiative in support of free and open-source alternatives to the X Desktop Group's systemd init system and middleware suite.

The purpose of the FreeInit.org initiative is not to build an alternative to the init system part of systemd; it is instead to act in support of projects aiming to build alternatives to systemd more broadly, now that systemd means much more than an init system.

Our initial goals are:

  1. Creating a list of systemd components, the FOSS alternatives to these, and the status and systemd-compatibility of these alternatives.
  2. Creating a libsystemd implementation providing the various utility libraries (e.g. libsystemd-bus), and with init, journal, and login management functionality abstracted so that it may work with alternatives.
    This will unify the disparate libsystemd implementations of e.g. Basu and elogind.
  3. Promoting the uptake of seatd/libseat, respectively a seat-management daemon and a library providing an abstract interface backed by either seatd or systemd-logind.
  4. Building solidarity between those who think free software is more than just a buzzword.

If you feel you can add to our project, please visit our GitHub repository and create an issue.
